    Hello World!

    Hi, this is my first blog post!

    In this blog you will find content related to my experience as software developer.

    Last year I had the opportunity to try and learn a lot of new things, like:

    • Reuse the business logic in C++ on iOS and Android using Djinni
    • New Software architectures (MVP and MVVM)
    • Fastlane, Jenkins and Travis-CI for Continuous Integration and Continuous Delivery among others, which I’m planning to write here.

    To do the things I wrote above, I received lot’s of help from the community like Fastlane, Djinni, some slack channels like ios.slack.com, androidtalks.slack.com, mobilecpp.slack.com, and that gave me motivation to give something back to the community.

    This blog is hosted on Github Pages and it’s built using Jekyll and Travis-CI, which I’m going to talk about on the next post.
